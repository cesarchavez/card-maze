Cookie Policy

How {{ app_title }} uses browser cookies to store your run data

What are cookies?

Cookies are small pieces of data stored in your browser by a website. This app uses a server-side session cookie — a single cookie that acts as a key to data held temporarily on the server while your browser window is open.

What data is stored?

The following information is kept in your session to make the game work across page navigations (for example, visiting a prize page and returning to the deck):

Data Purpose
Deck order The shuffled sequence of cards remaining in your current run
Current card The card drawn but not yet marked as completed, so it is restored if you navigate away
Scores Your accumulated points per suit (Hearts, Diamonds, Clubs, Spades, Joker)
Completed cards The list of card IDs you have finished in the current run, used for the share feature
Seed The daily shared seed (or a custom seed) that determines the deck shuffle for reproducibility
Username Your chosen display name within the app (stored only if a login system is enabled)
No tracking, no third parties

This app uses no third-party cookies, analytics, advertising trackers, or fingerprinting. The session cookie is strictly functional — without it, your progress would be lost every time you navigate to a different page.

How long is data kept?

Session data lives only as long as your browser session. Closing the browser tab or window (or clearing cookies) removes it. No run data is permanently stored on the server beyond what you explicitly save using the Save Run Results feature.

How to remove cookies

You can clear the session at any time by using your browser's Clear Cookies or Clear Browsing Data option for this site. Resetting the deck from the app menu will also start a fresh session.
